A) HTML Structure - top

To install Beloon on your web server, download the files from Templateforest and unzip them to the local disk on your computer. Inside the extracted folder you will see three folders named Beloon - Documentation. Beloon - Template and Custom Line Icons.

Beloon - Template folder structure is as follows:

  • css - contains all verndor and custom css files
  • fonts - contains icon fonts
  • img - contains all images used in the template
  • js - contains all vendor and custom js files
  • index.html - Home Page

Using an FTP Client (such as Filezilla or Total Commander), you will need to upload all of the contents of the Beloon - Template folder to your server. And other files can be used according to your preferences.

B) CSS Files and Structure - top

All the css files are located in the css folder of the template.
style.css - Main stylesheet which includes all the styles of the template.
colors/gradient/color.css - Stylesheet for color change of the template.

C) JavaScript - top

All vendor, 3rd party plugin and our custom JavaScript files are located in the js folder.
main.js - Our custom JavaScript file. You can add your custom js codes there if you need to.

Please DO NOT edit this files unless you have JavaScript coding experience. Simple mistakes can cause serious damage to the scripts

This Template imports three Javascript files.

  1. jquery
  2. jquery-migrate
  3. circliful
  4. bootstrap
  5. isotope
  6. slick
  7. tilt
  8. wow
  9. waypoints
  10. counterup
  11. main

Note: jQuery is a Javascript library that greatly reduces the amount of code that you must write.
